# Projeto
Banco de dados do mercado:
Requisitos:
1. Salvar os produtos cadastrados em um arquivo do HD ao finalizar o programa. Quando o
programa inicializar carregar esses produtos já cadastrados nas execuções anteriores. 5%
2. Menu diferenciado para cliente e funcionário.
1. Cliente:
1. Opções de busca: filtrar por preço, quantidade de estoque. 5%
2. Carrinho de compras: (mostrar) itens e preços, subtotal. Ter opção, inserir e retirar
item do carrinho, fechar compra. 10%
2. Funcionário:
1. Alertar estoque baixo e produto fora da validade. 5%
2. Entrada de produtos novos (pelo teclado ou por um arquivo). 5%
3. Reposição de estoques. 5%
4. Opções de busca: filtrar por preço, quantidade de estoque. (Igual ao do cliente)
3. Criar uma registro para armazenar informações de cada produto 5%
1. Campo categoria: Limpeza, alimento perecível, alimento não perecível, eletrônico
2. Campo código;
3. Campo preço;
4. Campo descrição
5. Campo fornecedor ou fabricante
6. Campo data de validade
7. Localização {corredor e prateleira}
4. Funções de busca e listagem:
1. Buscar por nome 5%
2. Buscar por fabricante ou fornecedor 5%
3. Ordenar por menor preço 5%
4. Ordenar por quantidade em estoque 5%
5. Ordenar por ordem alfabética 5%
6. Ordenar por categoria {alimentícia, limpeza,etc}5%
